# 🎫 Servicio de Eventos

Microservicio para la gestión de eventos del Sistema de Venta de Entradas.

## 🚀 Tecnologías

- **Rust** con **Actix-Web**
- **MySQL** con **SQLx**
- **JWT** para autenticación
- **Tracing** para logs

## 📦 Instalación

```bash
# Instalar dependencias
cargo build

# Configurar variables de entorno
cp .env.example .env

# Editar .env con tus credenciales
